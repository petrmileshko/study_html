# HTML_1

## Задание - создать Документ HTML с основными элементами структуры

1. Создать файл index.html
2. Первой строкой указать вид документа HTML
3. Далее создать корневой элемент html
4. В корневой html вложить два основных элемента head и body
5. В элемент head вложить элемент title
6. В элементе body вывести текст 'Привет мир!'
7. Запустить Live Server, чтобы сразу видеть результат в браузере.
